== Description ==

A specific plugin use in Upside Lite Theme, included some custom post types,widgets and shortcodes.

== Installation ==

1. Upload the files to the /wp-content/plugins/upside-lite-toolkit/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance->Theme Options and use.
